# DayCast 🌤️
 
DayCast is a weather lifestyle app that goes beyond raw forecast data. Rather than only showing showing temperature values and conditions, it translates the numbers into personalized recommendations that help users better prepare for their day.

## Live Demo
 
🔗 **[Try it out!](https://daycast-three.vercel.app/)**

## Features
 
- **Live weather data**: current conditions, 24-hour hourly forecast, and 7-day outlook
- **AI-powered suggestions**: recommendations generated from real-time weather data (what to wear, what to do, what to avoid, commute impact, health alerts)
- **Personalization**: user-configurable preferences including activity preferences, temperature sensitivities, health factors, and clothing style that feed directly into the AI's suggestions
- **Location search**: city search with autocomplete, plus one-tap "use my location" powered by geolocation + reverse geocoding
- **Unit preferences**: toggle between °C/°F and km/h/mph
